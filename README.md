# Latitude AI - Android APK

将 [dglatitude.cc](https://www.dglatitude.cc) 封装为原生 Android 应用。

## 📱 功能

- ✅ 原生 Android WebView，丝滑体验
- ✅ 全屏沉浸式加载 dglatitude.cc
- ✅ 支持 JavaScript 和本地存储
- ✅ 返回键页面回退
- ✅ arm64-v8a + armeabi-v7a 双架构

## 🚀 安装 APK

1. 前往 **Actions** 页面
2. 点击最新的 workflow run
3. 在 **Artifacts** 区域下载 `latitude-ai-apk.zip`
4. 解压得到 `.apk` 文件，安装到手机

## 🛠️ 技术栈

- **Android SDK 34** + **Java 17**
- **Gradle 8.5** 构建系统
- **GitHub Actions** 自动化构建
